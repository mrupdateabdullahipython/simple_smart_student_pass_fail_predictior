# 🎓 Smart Student Performance Predictor Dashboard

An advanced, interactive AI-powered web application built to predict student academic outcomes (Pass/Fail) using Machine Learning. This project leverages an optimized Logistic Regression model combined with a modern, glassmorphic Streamlit interface.

## 🚀 Key Features
- **Real-Time AI Inference:** Enter data dynamically to calculate passing probability instantly via an intuitive dashboard.
- **Interactive Visual Analytics:** Powered by Plotly Express to display distribution, performance correlation, and trends.
- **Model Diagnostics:** Built-in performance evaluation metrics (Accuracy, Precision, Recall, F1 Score) alongside a live Confusion Matrix.
- **Premium Cyber Dark UI:** Fully customized with custom CSS for a professional, modern user experience.

## 🛠️ Tech Stack
- **Language:** Python
- **Framework:** Streamlit (Custom CSS)
- **Machine Learning:** Scikit-Learn (Logistic Regression, StandardScaler)
- **Data Engineering:** Pandas, NumPy
- **Data Visualization:** Plotly Express & Graph Objects
